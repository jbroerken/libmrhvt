MRH_VOICE_TOOLS_LIB_VERSION_MINOR
=================================
The MRH_VOICE_TOOLS_LIB_VERSION_MINOR macro defines the minor library version.

Header
------
The macro definition can be found in the following file:

.. code-block:: c

    #include <libmrhvt/MRH_VTRevision.h>


Definition
----------
The defined minor version in the header is as follows:

.. code-block:: c

    #define MRH_VOICE_TOOLS_LIB_VERSION_MINOR 0
